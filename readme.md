# library manager app
## About
Library manager app(CRUD) that can help the librarian to manage the books in the table (name, author, price, date of publish, type). it can edit or delete any row in the table.  
## Built with 
- HTML5  
- CSS3  
- Javascript  
- Boostrap v5.
